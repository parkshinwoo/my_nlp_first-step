[What is this?]

마르코프 체인 알고리즘을 활용한 가사 생성 프로그램입니다. 
가사 수집 크롤러로 획득한 txt 파일을 사용하며
첫 실행시 json 파일이 생성됩니다. markov-장범준.json
이 json 파일은 가사 데이터의 전후 문맥 관계를 기록한 일종의 '사전'으로 가사 생성에 사용되는 핵심 데이터입니다.
json 파일이 생성된 이후에는 이를 기반으로 매 실행때마다 새로운 가사를 생성합니다.

[How to run]

python markov_lyric_generate.py 장범준

[Data]

가사 수집 크롤러의 실행결과로 생성되는 장범준.txt 파일을 사용합니다.
이 파일에서 각 곡의 마지막 문장은 ;(세미콜론)으로 종료됩니다.
이 세미콜론을 기준으로 한 곡, 한 곡이 나누어져서 마르코프 체인 알고리즘이 적용 됩니다.

[purpose?]

특정 가수 스타일의 가사를 매번 새롭게 작사하는 것을 목표로 했습니다.

[마르코프 체인?]

문장의 전후관계를 기반으로하여 확률적으로 문장을 생성하는 기법입니다.

1. 문장을 형태소 분석하여 품사별로 나눕니다. (명사, 동사 등)

2. 각 요소의 전후관계를 알 수 있게 3개의 요소 씩 묶어 사전에 등록합니다. 수많은 문장에 대해 수행하면 복잡한 구조의 사전을 얻을 수 있습니다.
   
   ex) 걱정|들|마세요
        마세요|매우|좋아
        |좋아|합니다|

3. 완성된 사전을 참고해 특정 단어에 대해 이어질 수 있는 단어를 확률적으로 선택합니다.
   
ex) 걱정:
    들: {마, 이제}
    마세요: {설마, 그게}
    마라: {괜찮다, 전혀}

4. 앞 단어와 이어질 수 있는 단어가 선택되면 그를 기반으로 이어가며 문장을 구성합니다.

ex) 걱정 마라 전혀


[Possible improvement]

2. word2vec 등의 기법으로 의미론적인 구조 형성
3. 사용자의 입력으로 특정 단어가 포함되면 그 의미를 반영한 가사 생성
   ex) '사람 + 사랑 + 여행' -> 두 연인이 떠난 여행에서의 사랑을 주제로 한 가사
4. LSTM을 적용하여 특정 가수 스타일의 가사를 생성
