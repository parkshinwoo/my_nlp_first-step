[What is this?]

네이버 뮤직에서 특정 가수의 전곡 가사를 수집하고 전처리를 수행하는 프로그램입니다.

[How to run]

수집하고자 하는 가수의 이름을 인자로 실행합니다.

python artist_lyrics_crawl_preprocessing_data_generation.py 장범준


[Purpose?]

자연어 처리를 위한 가사 데이터 수집
수집한 가사 데이터 전처리 (구둣점 제거, [Verse1] 등의 불필요한 태그 제거, 순수 한글 데이터를 위한 영어 제거(yeah 등의 불용어(stop words) 제거 목적도 겸함))
유사한 가사 추천 기능을 위한 엑셀 데이터 생성
가사 생성 기능을 위한 텍스트 파일 생성

[What is the result for running this program?]

1. 전처리를 완료한 가사 데이터를 엑셀로 저장합니다.
    -> 총 두개의 시트로 구성됩니다.
         a. [track_info] : 가수이름, 곡 제목, 트랙 ID(곡 구분자), 가사의 총 문장 수, 영어 포함 여부
         b. [Sentence_data] : 트랙 ID, 전처리된 가사(구둣점, 특수기호, 태그, 영어 제거된 순수 한글 가사), 라인 번호

2. 가사 생성 기능을 위한 텍스트 데이터를 저장합니다.
    -> 마르코프 체인을 통한 가사 생성 기능에 사용될 txt 데이터입니다.
        각 곡의 마지막 문장이 ;(세미콜론)으로 끝나게 됩니다.
